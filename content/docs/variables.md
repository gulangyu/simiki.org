## Site Variables ##

`site` : Configurations from `_config.yml`. Detailed description in [Configuration](/docs/configuration.html)

* `site.url`
* `site.title`
* `site.keyword`
* `site.description`
* `site.author`
* `site.root`
* `site.source`
* `site.destination`
* `site.themes_dir`
* `site.theme`
* `site.default_ext`
* `site.pygments`
* `site.debug`
* `site.index`

## Page Variables ##

`page` : configurations from a single wiki page's yaml header. Detailed description in [Metadata](/docs/metadata.html)

* `page.title`
* `page.category`
* `page.date`
* `page.content`
* `page.filename`
